Case Study: NYC 311 Data Analysis (Python)

1. Project Overview

Type: Data cleaning and analysis

Technologies: Python, Pandas, NumPy (optionally Matplotlib)

Purpose: Clean NYC 311 service request data and analyze request closing times.

2. Methodology

  • Load CSV data and inspect schema
  • Handle missing values; normalize categorical fields
  • Convert date columns and compute time deltas
  • Summarize by agency, complaint type, and borough

3. Tasks & Outputs

TaskOutput
Data CleaningNull handling and type conversions
Feature EngineeringClosing time metrics
AggregationTop complaint types and response times
ExportCSV summaries and optional plots

5. Outcome

  • Reproducible cleaning pipeline
  • Insights into closure times across categories
